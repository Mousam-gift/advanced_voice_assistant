🚀 Built an AI Voice Assistant using Python 

I recently developed a Voice Assistant that can perform multiple tasks using voice commands. This project helped me explore speech recognition, automation, and API integration while building an interactive assistant similar to basic versions of Siri or Alexa.

🔹 Features:
• Voice command recognition using SpeechRecognition 
• Weather updates using OpenWeather API 
• Send emails using voice commands (SMTP integration) 
• Play music or videos on YouTube 
• Google search using voice commands 
• Wikipedia-based knowledge responses 
• Reminder system with scheduling 
• Open applications and websites using voice 
• Time query and conversational responses 

🛠 Technologies Used:
Python, SpeechRecognition, OpenWeather API, Wikipedia API, SMTP, pywhatkit, schedule, webbrowser, dotenv

Through this project, I learned how to integrate multiple APIs, handle voice-based interactions, and design a modular assistant capable of automating everyday tasks.

I plan to improve this project further by adding Machine Learning-based intent classification and LLM integration for smarter conversations.
